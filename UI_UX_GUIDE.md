# 🎨 MoEatzy 디자인 및 UI/UX 가이드라인

본 문서는 MoEatzy 앱의 일관된 브랜드 아이덴티티와 냉장고 모티프의 UI를 구현하기 위한 스타일 규칙을 정의합니다. UI 에이전트는 컴포넌트 스타일링 시 본 문서를 준수하십시오.

## 1. 디자인 콘셉트 (Concept)
- **Tone & Mood**: 시원하고, 청결하며, 군더더기 없는 모던함.
- **Visual Motif**: 냉장고 문을 열었을 때 마주하는 신선하고 깨끗한 내부 공간의 인상. 유치한 만화풍 캐릭터를 지양하고, **백색 플라스틱 감성의 글래스모피즘(Glassmorphism)과 미세한 블러 효과**를 조화롭게 사용하여 고급스러운 테크 감성을 연출합니다.

## 2. 컬러 시스템 (Color Palette)
- **배경색 (Background)**: `#F4F7FA` (완전한 흰색이 아닌, 약간의 시원한 감도가 도는 연한 그레이시 블루)
- **주 메인 컬러 (Primary)**: `#E3F2FD` (신선함을 뜻하는 투명한 Ice Blue)
- **주요 텍스트 및 강조색 (Dark Navy)**: `#1A237E` (깊은 신뢰감과 가독성을 주는 딥 네이비)
- **D-day 경고 시스템 컬러**:
  - 위험 (D-1 이하): 배경 `#FFEBEE`, 텍스트 `#C62828` (소프트 레드)
  - 경고 (D-3 이하): 배경 `#FFF8E1`, 텍스트 `#F57F17` (소프트 옐로우)
  - 안전 (D-5 이상): 배경 `#E8F5E9`, 텍스트 `#2E7D32` (소프트 그린)

## 3. 컴포넌트 레이아웃 및 규격 (Layout Rules)
- **폼팩터 제약**: 발표 및 시연 시 완벽한 스마트폰 앱의 인상을 주기 위해 전체 컨테이너를 다음과 같이 고정합니다.
  - `max-width: 400px;`
  - `height: 100vh;`
  - `margin: 0 auto;`
  - `box-shadow: 0 20px 25px -5px rgb(0 0 0 / 0.1);` (공중에 떠 있는 모바일 기기 느낌 연출)
- **라운드 코너**: 모든 카드, 버튼, 팝업의 모서리는 부드러운 라운드값을 적용합니다 (`border-radius: 16px;` 또는 Tailwind `rounded-2xl`).
- **하단 고정 탭 바**: 화면 최하단에 항상 고정(position: fixed 또는 flex-end)되어 있어야 하며, 활성화된 탭은 딥 네이비 컬러와 하단 인디케이터 선으로 강조 처리합니다.
